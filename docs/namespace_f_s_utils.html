Detailed Description

Utility functions related to the file system.

FSUtils completely includes the older FileUtils namespace. In order to continue to use code written for the FileUtils namespace you need to define DEPRECATED_NAMESPACE at compile time. The definition will enable FileUtils by namespace FileUtils = FSUtils.
